# Contributing

1. Branch from `main` and keep changes focused.
2. Never commit secrets, models, generated media, or machine-specific paths.
3. Run pytest plus JavaScript and shell syntax checks.
4. Keep English and German locales in parity.
5. Document behavior, security impact, and validation in the pull request.

Pin new runtime dependencies and record third-party software in
`THIRD_PARTY_NOTICES.md`.
